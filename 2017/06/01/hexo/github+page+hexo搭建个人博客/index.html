    前言

    github page 是 github 的一项静态页面服务，自从这功能上线以来很多人利用这个功能来制作项目的介绍站点，但是也有很多人利用这个功能来制作自己的博客网站。只要利用 hexo 就能够很好地搭建自己的博客平台，而且 hexo 下拥有很多的主题插件让自己的博客添加更多的功能。而且 github page 提供了域名绑定，只要在 github page 设置中绑定自己的网站域名，就可以用自己的域名来访问这个博客了。

    github创建项目

    在 github 上创建 yourname.github.io 的仓库

    参考 hexo-generator-feed

    主题配置

    编辑主题配置文件，themes/indigo/_config.yml。

    默认配置如下

    添加新菜单项时，在 menu 下增加子属性即可。属性说明如下：

    rss

    6.x 后移除该配置项，使用站点配置中的 feed 作为统一配置。

    favicon

    站点 logo，显示在浏览器当前标签页左上角。

    头像

    位于左侧菜单上方

    email

    头像下方

    color

    设置 Android L Chrome 浏览器状态栏颜色，不需要可去除此项或设为 false。

    可以在 Markdown 文件中加 以分割摘要与文章正文。未设置时，按 excerpt_length 设置截取。
    文章摘要渲染方式: 为 true 时将渲染为 html，否则为文本
