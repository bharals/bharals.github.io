前言

我为什么要使用七牛云？

开始的时候，正在做自己的静态博客，用 hexo 的时候我一般都是直接在 markdown 里面插入本地的图片。这样做是很正常却也是很傻的一种方式。这样做的好处虽然是少了上传图片的这一个步骤，却是没有考虑到博客拥有很多图片时上传图片占用了很大的时间和空间。这样长远来看是很划算的一件事，于是我就去网上找了好些图床。

百度了好多教程，都是推荐七牛云的为多（主要是七牛免费，哈哈）。为了建立七牛图床，也是花了自己好大的时间。悲催。流泪…下面开始正题。

上传图片的多种方式

  1. 强烈推荐： http://mpic.lzhaofu.cn/ 用这个软件上传图片，下载用过就知道其方便。
  2. Google 插件上传： https://chrome.google.com/webstore/detail/emmfkgdgapbjphdolealbojmcmnphdcc
  3. 官方 qshell 上传下载，本人目前还没用，windows 下太不友好。闪退启动不了，我就能不了了之了。

感言

折腾这个七牛图床，花了好长时间，没想到结果还是比较理想的。至于用官方 qshell 工具上传下载同步以后有时间再说吧。未完待续。。。
