主题依赖安装

在 Hexo 根目录，如果以下插件已安装过，无需再次安装。

Less

主题默认使用 less 作为 css 预处理工具。

Feed

用于生成 rss。

Json-content

用于生成静态站点数据，用作站内搜索的数据源。

QRCode

用于生成微信分享二维码。

可选，不安装时会请求 jiathis Api 生成二维码。
